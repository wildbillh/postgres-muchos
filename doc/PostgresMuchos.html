query

The query event is emitted when the query method is called, but before the submission to the db. The object emitted has the process ID of the client and the sql. This is often useful for debugging purposes.
Type:
  • object
Properties:

parms array The paramters passed with the sql command.

results

The results event is emitted when the query method has resolved. Configuring this event could slow processing slightly. For performant applications, it should be turned off with emitControl.
Type:
  • object
